# webpack-static-s3
Bootstrap a static site generator using Webpack - deploy to S3

Use Webpack to build your static site, then deploy to S3!

Dev build
```
> yarn d
```

Dev server
```
> yarn s
```

Prod build
```
> yarn p
```

### Deploy to AWS S3

Copy `.env.example` to `.env` then add your AWS credentials and details

Upload to S3

```
> yarn u
```
